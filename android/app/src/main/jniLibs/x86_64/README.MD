# X86_64 (AMD64)
**Target:** Modern 64-bit Intel/AMD desktops and Emulators.

### Responsibility:
- The standard for high-performance Android Emulators (AVD) on modern PCs.
- Used for ChromeOS support and some specialized 64-bit Intel tablets.
- Provides the fastest development environment for Flutter by utilizing the host PC's full 64-bit power.
